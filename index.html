HTML

Aula 01 - Estrutura básica de uma página HTML

Este código apresenta a estrutura mínima de uma página web. Ele é a base para qualquer projeto front-end.

<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <title>Minha primeira página</title>
</head>
<body>
  <h1>Olá, mundo!</h1>
  <p>Esta é minha primeira página HTML.</p>
</body>
</html>

Git / GitHub

Aula 06 - Versionamento com Trunk-Based Development

Atividade acadêmica sobre integração frequente, branch principal, commits e feature flags.

ATIVIDADE - TRUNK-BASED DEVELOPMENT

Parte 1 - Estratégia de Branches
No Trunk-Based Development, utilizaremos principalmente a branch main.
As branches devem ser curtas e durar poucas horas ou no máximo um dia.

Parte 2 - Frequência de Integração
Ana e Bruno devem fazer commits frequentes durante o dia.
Ao final do expediente, devem integrar as alterações na branch main.

Parte 3 - Feature Flags
Nome das flags:
- habilitarPagamentoPIX
- habilitarCupomDesconto

Valor padrão: false

Parte 4 - Pseudocódigo
FUNÇÃO renderizarTelaDePagamento()
    mostrar título "Tela de Pagamento"

    SE habilitarCupomDesconto for verdadeiro ENTÃO
        mostrar campo "Cupom de Desconto"
    SENÃO
        não mostrar nada
    FIM SE

    SE habilitarPagamentoPIX for verdadeiro ENTÃO
        mostrar botão "Pagar com PIX"
    SENÃO
        mostrar botão "Pagar com Cartão"
    FIM SE
FIM FUNÇÃO.
