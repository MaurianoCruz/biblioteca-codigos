Biblioteca de Códigos

Material de apoio para os alunos copiarem e utilizarem nas atividades.

Orientação aos alunos:
Leia a explicação de cada atividade, copie o código indicado e cole no programa utilizado em aula. Lembre-se de testar, observar o resultado e tentar entender cada linha do código.
HTML

Aula 01 - Estrutura básica de uma página HTML

Este código mostra a estrutura inicial de uma página HTML. Ele serve como base para criar qualquer página da internet. 

<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <title>Minha primeira página</title>
</head>
<body>
  <h1>Olá, mundo!</h1>
  <p>Esta é minha primeira página HTML.</p>
</body>
</html>
CSS

Aula 02 - Estilizando uma página com CSS

Este exemplo ensina como mudar cor, fonte e espaçamento usando CSS. 

body {
  background-color: #f4f4f4;
  font-family: Arial, sans-serif;
}

h1 {
  color: blue;
  text-align: center;
}

p {
  color: #333;
  font-size: 18px;
}
JavaScript

Banco de Dados

Aula 05 - Criando uma tabela em SQL

Este código cria uma tabela simples de alunos em um banco de dados. 

CREATE TABLE alunos (
  id INT PRIMARY KEY AUTO_INCREMENT,
  nome VARCHAR(100) NOT NULL,
  email VARCHAR(100),
  turma VARCHAR(50)
);
